# 🦠 COVID-19 Data Visualization Dashboard

A comprehensive, interactive web application built with Streamlit for visualizing and analyzing COVID-19 data worldwide.  
This dashboard provides real-time insights into the pandemic's progression with beautiful, dark-themed visualizations.

## 🚀 Features

### 📊 Interactive Visualizations
- **Time Series Analysis**: Dynamic 2x2 subplot layout showing cumulative and daily cases/deaths
- **Country Analysis**: Comparative bar charts for top countries by various metrics
- **Interactive World Map**: Animated geographical visualization with time-based controls
- **Real-time Metrics**: Key performance indicators with recent trends

### 🎛️ Advanced Controls
- **Country Selection**: Search and filter by any country/region
- **Dynamic Filtering**: Adjust number of countries displayed in analyses
- **Map Customization**: Toggle between cumulative and daily cases on the map
- **Animation Controls**: Time-based map animation with play/pause functionality

### 📈 Key Metrics Tracked
- Total Confirmed Cases
- Total Deaths
- Total Recovered Cases*
- Active Cases
- Death Rate & Recovery Rate
- Recent 7-day trends
- Global Rankings

### 3. Country Analysis
